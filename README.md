# Giocatori

Template Next.js per **schede giocatore** dedicate a **procuratori e agenzie di basket**: profilo tecnico (scout view), statistiche, video, roster agenzia e contatti.

## Avvio locale

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) — la home reindirizza al profilo demo principale.

## Contenuti

I profili stanno in `data/athletes/` (uno file per slug). Aggiungi un nuovo modulo, esportalo in `data/athletes/index.ts`, e compilare con immagini in `public/athletes/`.
