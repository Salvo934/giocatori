# Istruzioni per gli agent (Cursor / AI)

Questo repository è un template **Next.js** (App Router) per **schede giocatore** basket pensate per **procuratori e agenzie** (player card).

- UI profilo: `components/profile/`
- Dati profilo: `data/athletes/*.ts`
- Tipi: `lib/types/athlete.ts`

Quando modifichi il codice:

- Allineati a pattern esistenti (`SectionShell`, Tailwind, tono UI già usato nel progetto).
- Non aggiungere documentazione markdown non richiesta.
- Comandi dev: vedi `README.md`.
